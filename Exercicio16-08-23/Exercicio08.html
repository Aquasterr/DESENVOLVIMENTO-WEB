Exemplo de Lista de Descrição

TCP
O Protocolo de Controle de Transmissão (TCP) é um método (protocolo) usado junto com o Protocolo da Internet (IP) para enviar dados na forma de unidades de mensagem, chamadas de pacotes, entre computadores pela INternet.
IP
O Protocolo da Internet (IP) é o método ou protocolo pelo qual os dados são enviados de um computador para outro na Internet. Cada computador na Internet é indentificado de forma unica por um endereço IP
FTP
O protocolo de Transferencia de Arquivos (FTP) é um protocolo usado para trocar arquivos entre computadores na INternet.
HTTP
O Protocolo de Transferencia de Hipertexto (HTTP) é o protocolo usado para trocar texto, imagens graficas, som, video e outros arquivos multimidia na web
